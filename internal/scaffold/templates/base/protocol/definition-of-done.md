# Definition of Done (DoD)

Mặc định cho mọi ticket. Reviewer gác đúng checklist này; ticket có thể thêm DoD riêng.

- [ ] Code khớp scope ticket, không đụng file ngoài `context_refs` chưa khai.
- [ ] Test liên quan pass.
- [ ] Lint/format sạch.
- [ ] Diff review bởi agent ≠ implementer (no-self-review).
- [ ] Ledger có entry hoàn thành + handoff note.
- [ ] Tri thức đáng giữ đã ghi KB (nếu ticket nhiều tri thức).
- [ ] Branch có PR (human gate cho merge).
