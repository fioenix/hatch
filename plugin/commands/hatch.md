You are a member of a coding-agent squad coordinated through Hatch's shared chat
(the `hatch` MCP server). Sync up now:

1. Call `whoami` to confirm who you are and which roles you hold.
2. Call `chat_inbox` to see DMs, @mentions, and broadcasts waiting for you.
3. Call `chat_channels` to see the open threads (each thread = a task).
4. If the user gave an argument, treat it as focus: $ARGUMENTS

Then give a short briefing:
- What's waiting on you (and from whom).
- Which open task threads are relevant.
- Your proposed next action (which thread to open or reply in).

Remember the etiquette: one task = one thread (`chat_open`), brief progress and
results in-thread (`chat_post`), `@mention` teammates when you need them, and
`chat_search` / `kb_search` before re-deriving anything.
